html 태그 구분

시작과 끝나는 태그로 묶인 쌍으로 이루어진 태그, 단일 태그

block 속성/inline 속성

콘텐츠 타입의 태그 : 콘텐츠 용도에 따라서 태그를 분류

block 속성

문단의 성격을 가진 태그

종료 태그가 있따면 종료 태그 후 자동으로 줄을 바꿈 즉 엔터

정렬이 가능하고 너비(width)와 높이(heigth)를 지정할 수 있음

block형 속성의 태그는 기본적으로 block형 속성의 태그를 포함할 수 없음(예외 존재)

block형 속성의 태그는 inline형 속성의 태그를 포함할 수 있음

inline 속성

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nesciunt provident voluptatum veritatis enim consequuntur velit animi eligendi saepe. Commodi, ratione rerum soluta aliquid minima hic illo tenetur repellat ullam perspiciatis.

inline의 경우 문자와 같은 속성으로 자동으로 줄바꿈을 해주지 않음

한 문자만 정렬하는 것이 아님 -> 문단을 기준으로

문자의 크기를 변경하지만 너비와 높이를 지정하는 것은 아님

inline형 속성의 태그는 inline형 속성의 태그를 포함할 수 있음

태그를 중첩해서 사용할 때는 중첩이 꼬이지 않도록 주의할 것
