자바스크립트 특징

  • 객체지향적 프로그래밍 언어(bom, dom)
  • 프로그래밍 언어 : 변수, if, for, function
  • html, css 조작 가능한 언어
  • server 제작할 수 언어
  • script 태그 내에서 실행 가능

프로그램 언어 처리 순서

입력(input) -> 처리(process) -> 출력(output)

출력

  • 화면단에 출력
    document.write('본문 아래 쓰기');
  • 메시지창 띄우기
    alert('메시지 창 띄우기');
  • 에러 확인창
    f12 > console 탭 > window.console.log('error 메시지');
