js 데이터

숫자, 문자, 논리형, undefined, NaN, Null, Array, function, objec, class ... 변수 : 데이터를 저장하는 공간(메모리)
  • 첫글자는 숫자를 제외한 나머지 언어로 시작
  • var 전역변수 처럼 사용 : 같은 이름의 변수를 여러번 선언해도 에러가 발생하지 않음
    프로젝트가 커지면 중복 현상으로 오류를 해결하지 못할 수도 있음
  • let : var 문제를 해결하는 방법
    변수를 중복 선언시 오류 발생
  • const : 상수선언
  • const 남자 = true;
  • const 여자 = false;
  • 변수명을 한글로 사용할 수 있으나 사용하지 않음
