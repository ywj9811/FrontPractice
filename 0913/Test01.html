HTML이란?

html은 Hyper Text Markup Language
       구조를 이쁘게 만들어도 페이지 이쁘지 않음

CSS란?

css는 Cascading style sheet
       이는 디자인을 입히는 언어로 이쁘게 만들 수 있음
       이 두가지 언어는 조립해서 사용하는 것이 가능한 언어임

JavaScript란?

javaScript는 논리적 사고가 필요한 언어로 데이터를 활용할 수 있는 언어임

페이스북(메타)에서 만든 React(라이브러리) : html + css + javascript + xml

   

** html의 특징 **

   백색문자(white space) 즉 enter와 2개 이상의 띄어쓰기는 인식하지 못함
   대소문자 구분되지 않음 하지만 소문자로 작업하는 것이 정석임
   태그의 경우 블록형 태그와 인라인 태그로 구분을 함 -> 한 줄을 통으로 먹거나 자신의 크기만큼만 차지하는가
   즉 문단으로 처리되거나 문자로 처리되는 것임
